Core Service

OverlayService

A lightweight overlay manager that tracks open overlay refs, locks body scroll while any overlay is open, and lets the topmost overlay be closed via closeTop() — wired to ESC on this page.

Stacking — multiple overlays

Each push adds a new ref to the stack. closeTop() removes only the last-registered overlay; ESC does the same.

Scroll locking

register() sets document.body.style.overflow = 'hidden' when the first overlay opens. deregister() restores it once the stack is empty. Open an overlay above and try scrolling this page — the page won't scroll while a panel is open.
